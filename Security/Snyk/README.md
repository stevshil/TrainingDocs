# Snyk validation

This folder is about getting Snyk to find issues and using Snyk in VS Code.

You should do the following in this folder before working.

```sh
git clone https://github.com/snyk-labs/nodejs-goof
```

This project has lots of issues, and makes it nice to work with Snyk to find issues.

You might need to set Snyk plugin to manual not automatic, in the settings.
